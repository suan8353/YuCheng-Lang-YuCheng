## 1. 概述

### 核心特性

- 📝 **提交管理** - 暂存、提交、修改提交

- 🔄 **同步操作** - 推送、拉取、获取
- 🔍 **历史查看** - 提交历史、文件历史
- ⚠️ **冲突解决** - 可视化冲突解决工具

## 2. 使用方法

### 2.1 IDE中使用

1. 打开"Git"面板
2. 查看文件变更
3. 暂存文件
4. 输入提交信息
5. 点击"提交"

### 2.2 常用操作

```bash
# 初始化仓库
yucheng git init

# 提交更改
yucheng git commit -m "提交信息"

# 推送到远程
yucheng git push origin main

# 拉取更新
yucheng git pull origin main

# 创建分支
yucheng git branch feature-new

# 切换分支
yucheng git checkout feature-new
```
